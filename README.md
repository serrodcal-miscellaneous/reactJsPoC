# Installing

```bash
~$ npm install -g create-react-app
~$ create-react-app songs-finder
~$ cd songs-finder
~$ npm start
```

**Note**: Maybe you need to update npm, run `npm install -g npm` to update!

You should have a `songs-finder` in your local as given below:

```
.
├── README.md
├── node_modules
├── package-lock.json
├── package.json
├── public
└── src
```
